### Student Features

- Upload resume (PDF or Image)
- Automatic text extraction (OCR)
- Paste any job description
- AI-based resume vs job matching
- Match percentage calculation
- Missing skill identification
- Simple explanations for each missing skill
- Role-specific interview preparation insights

### Admin / Placement Features
- Centralized skill gap storage
- Batch-level skill gap analysis
- Department and year-wise analytics
- Skill trend detection
- Recruiter-specific readiness analysis
- Curriculum feedback support
- Training and workshop planning
- Interactive analytics dashboard

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **AI**: Google Gemini API
- **Database**: SQLite
- **OCR**: Tesseract.js
- **Charts**: Chart.js

---

## Requirements

- Node.js **v18 or higher**
- npm
- Google Gemini API Key
